Facial Recognition Backend
Node.JS, Express APIs, Hosted on Heroku

Getting Started
`npm install`
Ensure logged into heroku `heroku login` 
Run dev server with `node server.js` defaults to port 3000

Prerequisites
Node.js 8.12
NPM 6.4.1

Built With
Node.js
Express
Fetch API
Heroku CLI
Cors
Knex
bCrypt
